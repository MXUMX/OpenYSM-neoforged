A more compatible version of Yes Steve Model

[https://github.com/OpenYSM/OpenYSM](https://github.com/OpenYSM/OpenYSM)

OpenYSM is a player model modification mod that provides the same functionality as YSM while solving its compatibility issues.

This mod is compatible with **almost all platforms**, including **macOS**, RISC-V, and even mobile phones.

The original mod is [https://modrinth.com/mod/openysm](https://modrinth.com/mod/openysm)
